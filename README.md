# LifeVault

**Your Personal Asset & Home Management Center**

Track everything you own, manage warranties, schedule maintenance, and generate insurance reports — all in one place.

## Features

- **Dashboard** - Asset overview, value tracking, warranty alerts, maintenance reminders
- **Inventory Management** - Add items via AI photo recognition, barcode scanning, or manual entry
- **Warranty Tracker** - Monitor warranty status, get expiry alerts
- **Maintenance Scheduler** - Recurring task management with priority levels
- **Insurance Reports** - Generate CSV reports for insurance claims
- **Dark Mode** - Easy on the eyes, day or night
- **PWA Ready** - Install as a mobile app, works offline
- **Responsive** - Works on desktop, tablet, and mobile
- **Data Export/Import** - Backup and restore your data

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- PWA (Progressive Web App)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
